### Information
A Mini E-Commerce Frontend Demo using Vue.js 3.

*This demo was made for the ALDI team, as my application for the `AI Empowered Frontend Developer Trainee` position.*

#### Used technologies
- Vue.js 3 - 3.5.30
- Vite - 7.3.1
- Typescript - 5.9.3
- Tailwind css - 4.2.1

### Setting up & Running the demo
```sh
npm install
npm run dev
```
